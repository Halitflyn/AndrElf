# 🌲 AndrElf - Культ Психо-Андрія

Інтерактивний веб-додаток, присвячений культу Психо-Андрія (Ельфа Асасіна з Дібрівських лісів). Проєкт поєднує в собі містичну атмосферу, тематичний музичний плеєр та звід священних правил.

## ✨ Особливості (Features)

- **🎵 Кастомний музичний плеєр**: Відтворення тематичних треків (Дарк Фолк, Шансон, Репчик), керування відтворенням (Play/Pause, Skip, Shuffle, Repeat).
- **❤️ Система вподобань**: Можливість ставити лайки та дизлайки трекам. Дані зберігаються локально у вашому браузері (`localStorage`).
- **👁️ Інтерактивний UI**: Містичні очі, які слідкують за курсором миші, та спеціальні ефекти при наведенні на "чай".
- **📜 Священні правила**: Звід правил культу з інтерактивними підказками та анімаціями (чайна пара, шепіт).
- **📱 Адаптивний дизайн**: Повна оптимізація інтерфейсу для мобільних телефонів та ПК.

## 🛠️ Технології (Tech Stack)

- **Фреймворк**: [React 19](https://react.dev/)
- **Збірка**: [Vite](https://vitejs.dev/)
- **Стилізація**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Мова**: TypeScript
- **Іконки**: [Lucide React](https://lucide.dev/)

## 🚀 Як запустити локально

1. Склонуйте репозиторій:
   ```bash
   git clone https://github.com/Halitflyn/AndrElf.git
   ```
2. Перейдіть у папку проєкту:
   ```bash
   cd AndrElf
   ```
3. Встановіть залежності:
   ```bash
   npm install
   ```
4. Запустіть сервер для розробки:
   ```bash
   npm run dev
   ```
5. Відкрийте посилання (зазвичай `http://localhost:3000` або `http://localhost:5173`) у вашому браузері.

## 🌐 Деплой (GitHub Pages)

Цей проєкт налаштований для автоматичного розгортання на GitHub Pages за допомогою GitHub Actions. 
Щоб деплой працював коректно, переконайтеся, що:
1. У файлі `vite.config.ts` вказано правильний базовий шлях: `base: '/AndrElf/'`.
2. У налаштуваннях репозиторію (Settings -> Pages) джерелом (Source) вибрано **GitHub Actions**.
